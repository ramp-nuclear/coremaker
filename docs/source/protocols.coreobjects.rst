Core Objects
============
This section deals with the protocols for cores and the objects they refer to.
These include the following concepts:

* Cores are objects that allow most packages to get components from the grid and
  the ex-grid elements, as well as populate the core grid with elements.
* Elements are objects that can yield components for adapters to work with.
* Components are the simplest object that make up a definition of a complex
  body. They have a geometry and the material within it.
